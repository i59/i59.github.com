CryMono

CryMono is an CryENGINE extension giving the power of C# into the hands of developers, allowing game logic to be scripted in a faster and more easily used language than the built-in alternative, lua.

As the lead developer, I have created the full C++ backend, and co-created the C# implementation along with one other programmer.

In its current state, CryMono allows for entities, flow nodes and game rules to be loaded into the engine and properly registered on startup in the engine. The currently being worked on update 0.4 includes epic additions such as realtime scripting, making sure scirpts are reloaded directly after having been edited, allowing for a true LiveCreate-like developer experience.

CryMono can be viewed at GitHub here, including full source to both CryMono.dll and CryBrary.dll as well as our two game samples.
