    CryMono Developer tool (CryENGINE3)
    View on GitHub

    CryMono is a Mono integration for CryENGINE3 that adds full .NET support to the game engine, allowing game logic to be scripted in C# and other languages as an alternative to the currently used Lua.

    Using CryMono, developers can create custom entities, game rules, flow nodes, UI setups and more. Scripts are also reloaded at runtime when edited, allowing for a truly intuitive developer experience.

    C# Entity Sample

    The code sample below demonstrates how easy it is to create an CryENGINE entity support model loading and physicalization. Simply drop the script into Game\Scripts\Entities and it will be automatically loaded and registered on engine startup.

    View Code

    using CryEngine;

namespace CryGameCode.Entities
{
    [Entity(Category = "Physics")]
    public class Rigidbody : Entity
    {
        [EditorProperty(Type = EntityPropertyType.Object)]
        public string Model { get { return GetObjectFilePath(); } set { LoadObject(value); } }

        [EditorProperty]
        public float Mass { get { return Physics.Mass; } set { Physics.Mass = value; } }

        protected override void OnReset(bool enteringGame)
        {
            Physics.Type = PhysicalizationType.Rigid;
        }
    }
}
